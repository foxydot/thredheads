== Description ==

This Plugin converts products, product categories, and more from WP E-Commerce to WooCommerce.

Note: It exports most of WP E-Commerce data except product variations. We are working on the best way to convert variations from WP E-Commerce to WooCommerce.

See http://www.woothemes.com/extension/

== Installation ==

1. Upload the folder 'woocommerce-wpec-converter' to the '/wp-content/plugins/' directory

2. Activate 'WooCommerce - WP E-Commerce Converter' through the 'Plugins' menu in WordPress

== Usage ==

1. Follow the installation steps above

2. De-activate WP E-Commerce

3. Activate WooCommerce

4. Open Tools > Import > WP E-Commerce To WooCommerce Converter

5. Click 'Convert Now' button

That's it!
